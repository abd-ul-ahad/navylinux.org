# navylinux.org

Navy Linux webpage, built using [Hugo](https://gohugo.io) and the theme [Doks](https://github.com/h-enk/doks).

## Requirements

- [Hugo](https://gohugo.io)
- [NodeJS](https://nodejs.org)

## Installing Dependencies

```bash
npm install
```

## Start the Development Server

```bash
hugo server
```

## Build the website

```bash
hugo
```
