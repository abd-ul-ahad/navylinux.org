Why Navy ?

Navy Linux and The Navy Linux Project is an on-going community project founded by Navy Foundation on January 4, 2021. The project aims to offer a minimalistic installation of the Linux operating system. Navy Linux is based on Redhat Enterprise Linux and shall always remain open to the community.

We promise the same experience that was the hallmark of the Minimal CentOS Linux
